## Prerequisites

- **Go 1.25+** with CGO enabled
- **Node.js** and npm (for the Svelte frontend)
- **SQLite3** development headers (for the Go sqlite3 driver)

## Clone and Build

```bash
git clone https://github.com/wesm/agentsview.git
cd agentsview
```

Build the full application (frontend + Go binary):

```bash
make build
```

This runs `npm install` and `npm run build` for the frontend,
then compiles the Go binary with the embedded SPA.

For a release build with optimizations and stripped symbols:

```bash
make build-release
```

## Development Workflow

Run the Go server and Vite dev server separately for hot
reloading:

```bash
# Terminal 1: Go server (API only)
make dev

# Terminal 2: Vite dev server (frontend with HMR)
make frontend-dev
```

The Vite dev server proxies API requests to the Go server.

## Make Targets

| Target | Description |
|--------|-------------|
| `build` | Build with embedded frontend |
| `build-release` | Release build (optimized, stripped) |
| `install` | Build and install to `~/.local/bin` or `GOPATH` |
| `dev` | Run Go server for development |
| `frontend` | Build frontend SPA only |
| `frontend-dev` | Run Vite dev server with HMR |
| `test` | Run all tests (`CGO_ENABLED=1 -tags fts5`) |
| `test-short` | Run fast tests only (`-short` flag) |
| `e2e` | Run Playwright end-to-end tests |
| `vet` | Go vet analysis |
| `lint` | golangci-lint |
| `tidy` | Tidy `go.mod` |
| `release` | Build for current platform |
| `release-darwin-arm64` | Cross-compile for Mac ARM |
| `release-darwin-amd64` | Cross-compile for Mac Intel |
| `release-linux-amd64` | Cross-compile for Linux x86_64 |
| `install-hooks` | Install pre-commit git hooks |

## Testing

Run the full test suite:

```bash
make test
```

Tests require CGO and the `fts5` build tag:

```bash
CGO_ENABLED=1 go test -tags fts5 ./...
```

Run only fast unit tests:

```bash
make test-short
```

Run Playwright end-to-end tests:

```bash
make e2e
```

## Project Structure

```
cmd/agentsview/         CLI entry point, server startup
internal/
  config/               Config loading, defaults, migration
  db/                   SQLite operations, schema, FTS5
  parser/               JSONL parsers (Claude, Codex, Gemini)
  server/               HTTP handlers, SSE, middleware
  sync/                 File watcher, discovery, worker pool
  timeutil/             Timestamp parsing
  web/                  Embedded frontend (embed.FS)
frontend/               Svelte 5 SPA source
scripts/                Build and test utilities
```

## Build Tags

| Tag | Required | Purpose |
|-----|----------|---------|
| `fts5` | Yes | Enables SQLite FTS5 full-text search |

The `fts5` tag must be passed to all Go commands:

```bash
go build -tags fts5 ./cmd/agentsview
go test -tags fts5 ./...
```
