## Quick Start

```bash
curl -fsSL https://agentsview.io/install.sh | bash
```

**Windows:**

```powershell
powershell -ExecutionPolicy ByPass -c "irm https://agentsview.io/install.ps1 | iex"
```

The installer detects your OS and architecture, downloads the latest
release from GitHub Releases, verifies the SHA-256 checksum, and
installs the binary.

```bash
agentsview              # Start server, open browser
agentsview -port 9090   # Custom port
agentsview -no-browser  # Start without opening browser
```

## What It Does

agentsview reads the session files that Claude Code, Codex, and
Gemini CLI leave on your machine and gives you a web interface to
work with them. It runs locally — nothing leaves your machine.

## How It Works

<img src="/architecture.svg" alt="agentsview architecture: agent sessions sync into SQLite with FTS5 search, served via REST API, SSE events, and embedded Svelte SPA" style="width: 100%; max-width: 960px; margin: 1.5rem auto; display: block;" />

agentsview watches your agent session directories for changes,
parses JSONL files from each agent format, and stores structured
data in SQLite with full-text search indexes. The embedded web
frontend provides browsing, search, and analytics over the
REST API.
